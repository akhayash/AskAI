### ステップ 2: Web UI を開く

ブラウザで以下を開く：
```
http://localhost:5000/ui/
```

### ステップ 3: エージェントを選択

左側のサイドバーからエージェントをクリック：

- **Contract Agent** - 契約関連の質問
- **Spend Agent** - 支出分析の質問
- **Negotiation Agent** - 交渉戦略の質問
- **Sourcing Agent** - 調達戦略の質問
- **Knowledge Agent** - 知識管理の質問
- **Supplier Agent** - サプライヤー管理の質問
- **Legal Agent** - 法務関連の質問
- **Finance Agent** - 財務関連の質問
- **Procurement Agent** - 調達実務の質問
- **Assistant** - 総合的な質問

### ステップ 4: 質問を入力

テキストボックスにメッセージを入力：

**例:**
```
新規サプライヤーとの契約で注意すべき点は？
```

Enter キーを押すか、「送信」ボタンをクリック。

### ステップ 5: 応答を確認

エージェントからの応答が表示されます。会話を続けることもできます。

## サンプル会話

### Contract Agent との会話例

**You:** 新規サプライヤーとの契約で注意すべき点は？

**Agent:** 新規サプライヤーとの契約では以下の点に注意が必要です：

1. **契約条項の明確化**
   - 納期、品質基準、支払条件を具体的に記載
   - 違約金条項の設定

2. **リスク管理**
   - 知的財産権の保護
   - 機密保持契約の締結

3. **実績確認**
   - 過去の取引実績
   - 財務状況の確認

より詳細な情報が必要でしょうか？

### Negotiation Agent との会話例

**You:** 価格交渉のコツは？

**Agent:** 効果的な価格交渉のポイント：

1. **事前準備**
   - 市場価格の調査
   - 競合他社の見積もり収集

2. **交渉戦略**
   - Win-Win の関係構築
   - 長期的パートナーシップの提案

3. **代替案の準備**
   - 数量や納期での調整案
   - 複数年契約での割引交渉

## トラブルシューティング

### Q: エージェント一覧が表示されない

**A:** DevUIHost が起動しているか確認：
```bash
curl http://localhost:5000/
```

成功すると JSON でエージェント一覧が返ります。

### Q: メッセージが送信できない

**A:** 以下を確認：
1. エージェントを選択しているか
2. Azure OpenAI の環境変数が設定されているか
3. `az login` で認証済みか

### Q: 応答が返ってこない

**A:** ブラウザの開発者ツール (F12) でコンソールエラーを確認。
Azure OpenAI のクォータやネットワーク接続を確認。

## ショートカット

- **Enter**: メッセージ送信
- **Shift + Enter**: 改行
- **Esc**: 入力欄をクリア (開発予定)

## ヒント

💡 **会話履歴**: エージェントを切り替えると会話がリセットされます  
💡 **複数行入力**: Shift+Enter で改行できます  
💡 **専門性**: 各エージェントは特定の分野に特化しています  
💡 **組み合わせ**: 複数のエージェントに同じ質問をして比較も可能
