## 主な機能

### 1. エージェント選択 (サイドバー)
- 10種類のエージェントがカード形式で表示
- 各カードには名前と説明を表示
- クリックでエージェントを選択
- 選択中のエージェントはハイライト表示

### 2. チャット機能
- ユーザーメッセージ: 右側、紫色の吹き出し
- エージェント応答: 左側、白色の吹き出し
- システムメッセージ: 黄色の背景

### 3. メッセージ入力
- 複数行テキスト入力に対応
- Enter キーで送信 (Shift+Enter で改行)
- 送信中は入力を無効化
- ローディングアニメーション表示

### 4. 会話履歴
- 会話がスクロール可能
- エージェント切り替え時にリセット
- マルチターン会話に対応

## デザイン特徴

### カラースキーム
- **プライマリ**: 紫 (#667eea) - ヘッダー、ボタン、ユーザーメッセージ
- **セカンダリ**: 緑 (#10b981) - エージェントアバター
- **背景**: グラデーション (#667eea → #764ba2)
- **カード**: 白 + シャドウ

### アニメーション
- 接続ステータスのパルス効果
- ローディングドットのバウンス
- ホバー時の色変化とトランジション
- メッセージ送信時のスムーズスクロール

### レスポンシブ
- 最小幅: 280px (サイドバー)
- 最大幅: 1200px (コンテナ)
- メッセージ幅: 70% (最大)

## 使用技術

- **HTML5**: セマンティックマークアップ
- **CSS3**: Flexbox、グラデーション、アニメーション
- **JavaScript (ES6+)**: Fetch API、async/await
- **依存関係**: なし (Pure Vanilla JavaScript)

## ブラウザ対応

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## セキュリティ考慮事項

- XSS対策: `.textContent` 使用 (一部 `.innerHTML` は制御済み)
- CORS: DevUIHost 側で設定済み
- 認証: 未実装 (開発環境用)

## パフォーマンス

- 初回ロード: < 1秒
- エージェント一覧取得: ~100ms
- メッセージ送信: Azure OpenAI のレスポンスタイムに依存
- メモリ使用: ~10MB (会話履歴含む)
